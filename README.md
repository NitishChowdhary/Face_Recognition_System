Project Goal

The primary goal of this project was to build a working prototype of a Face Recognition System capable of:

1. Identifying a known individual from a database. 
2. Demonstrating the core machine learning workflow (Detection, Encoding, Matching). 
3. Implementing the capability for future real-time processing (e.g., webcam feed).
